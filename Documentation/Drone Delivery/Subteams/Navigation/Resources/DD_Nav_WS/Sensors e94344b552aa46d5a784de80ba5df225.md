# Sensors:

GPS:

A good note for the gps sensor is that if you echo the topic using ROS it might look like it is not publishing the latitude and longitude values, because they barely change; however, this post has a good explanation of why that assumption is false: [https://github.com/tu-darmstadt-ros-pkg/hector_gazebo/issues/62](https://github.com/tu-darmstadt-ros-pkg/hector_gazebo/issues/62). Essentially, since the gazebo world is acting like it is taking a section of the earth, moving a couple of meters in any direction in x or y will not change the longitude or latitude that much as you would need to move multiple kilometers in any direction to get even 1 degree of change in longitude or latitude.
