# Accessing

1. Create an Authentik account: [Creating an account](https://www.notion.so/Creating-an-account-000e9732a0c54dee8690d7f94b1d92d4?pvs=21) 
2. Ask Josh or Seth to add you to the drone-delivery Authentik group (see [How to add users to a group](https://www.notion.so/How-to-add-users-to-a-group-65d2ed6f82f44b06bce9312141a0918c?pvs=21))
3. Follow ALL of these instructions to get the OpenVPN client and download the access file:  [OpenVPN Installation](https://www.notion.so/OpenVPN-Installation-d580a6ca75934fb58612475624131c45?pvs=21) 
4. Once connected to the VPN, go to [https://192.168.11.200:8006](https://192.168.11.200:8006/) to access Proxmox. NOTE: This IP may change. i setup a static IP, however it’s not guarenteed it will stay there. To find the ip, just login to the router config page at [http://192.168.11.1/Status_Lan.asp](http://192.168.11.1/Status_Lan.asp) with username / password: admin / password and scroll down and find the IP address of the DHPC client named “dc20-Standard-PC-i440FX-PIIX-1996”.
5. Proceed to the unsecure website (just allow it because your connection is secure over https)
6. Login with credentials from Authentik and set `purduearc.com` as the Realm
7. Press OK to the `No valid subscription` warning. You can ignore this.
8. View VMs by clicking pve0 dropdown on the left

## Creating a ROS VM

Create a Linked Clone VM from the VM template `112 (dd-ros-1)` .

![Untitled](ROS%20VMs/Untitled.png)

A LinkedClone only adds storage to the server that you change in the cloned VM. This helps save storage. You can see the total storage used by the ROS VMs by clicking on `vmdata-thin`  on the left panel.
