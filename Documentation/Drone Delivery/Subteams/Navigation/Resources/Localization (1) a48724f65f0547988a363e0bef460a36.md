# Localization (1)

When a robot is moving around, it needs to determine where it is in its environment. However, it needs to determine where it is/how it is moving both in its local and global environments. There are many different algorithms through which a robot can determine each of these, but in real solutions to these problems, multiple algorithms are applied simultaneously and then fused together by an extended Kalman filter(which is called sensor fusion) to produce a more accurate local and global estimate of the pose(position and orientation) of the robot. Using an EKF to fuse the algorithms produces a more accurate result as typically, these algorithms compensate for each others weaknesses/their errors partially cancel out, leaving a less erroneous result.
