# Drone Operation Notes

Facility Maps are a data source for permitted flying areas: [https://www.faa.gov/uas/commercial_operators/uas_facility_maps](https://www.faa.gov/uas/commercial_operators/uas_facility_maps)

When operating in controlled airspace (blue boxes, on Facility Maps) you need LAANC authorization: [https://www.faa.gov/uas/getting_started/where_can_i_fly/airspace_restrictions/flying_near_airports](https://www.faa.gov/uas/getting_started/where_can_i_fly/airspace_restrictions/flying_near_airports)

You can get a LAANC through here: [https://airspacelink.com/](https://airspacelink.com/)

Uncontrolled airspace not LAANC not required.

If below recommended ceiling on Facility Map LAANC is usually automatically approved.

Autonomous operations not clearly researched yet.

For flying beyond line of sight we need to fill out form: [https://www.faa.gov/uas/advanced_operations/instructions-drone-operators-completing-faa-form-7711-2](https://www.faa.gov/uas/advanced_operations/instructions-drone-operators-completing-faa-form-7711-2)

For package delivery we need: [https://www.faa.gov/uas/advanced_operations/package_delivery_drone](https://www.faa.gov/uas/advanced_operations/package_delivery_drone)
