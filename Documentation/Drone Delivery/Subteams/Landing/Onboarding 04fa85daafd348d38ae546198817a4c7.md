# Onboarding

As this is a relatively new sub-team, there is not much to catch up on. However, the advice is to go through [Resources](Resources%204ebc82092a3f465fbbb3fbadd0468c66.md)  to familiarize yourselves with:

- Current camera
- April tags
- Package container design
- Overall landing solution
- Top-down custom object detection model

Once you have done this, there are a few active areas to work on:

1. PNC the container (work closely with hardware)
2. Improve the object detection model (get more data, fine-tune training, etc.)
3. Explore 1d IR sensors and integration into the landing solution
4. Complete the landing solution (integration of localization from Navigation, landing container, object detection with camera mapping out local area through 360, and flat landing zone detection.)

Once you are done, go to ‣ and find a task/add a task you want to work on (communicate with sub-team lead for clarity.)
